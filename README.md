# TCC-Scanner

Collaborated with Dylan with this project, Github username: DTcCyber


TCC-Scanner is a technical tool designed to streamline the analysis of Lynis security audits. It takes raw .log files generated by system audits and uses Python-based parsing to categorize findings into actionable security domains.

Key Features:

- Automated Log Parsing: Utilizes Python (HTML.py and JSON.py) to scan raw logs for critical security metrics like the Hardening Index, Tests Performed, and Plugin Activity.

- Smart Categorization: Automatically groups security findings into seven essential domains:
SSH Configuration (Root login status, authentication methods)
Firewall Status (UFW/iptables activity)
Account & Password Security (Default users, password policies)
Filesystem Integrity (World-writable files, sensitive permissions)
Logging & Auditing (auditd/syslog status)
